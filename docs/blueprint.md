# **App Name**: G-Matrix: Gluten-Free Vibe Check

## Core Features:

- Product Search: Google-style search bar to find gluten-free products.
- Matrix Chart Display: Display a matrix chart (scatter plot) showing the 'Safety' vs. 'Taste' rating of a product. Leverage user votes aggregated with distributed counters.
- User Voting (Safety): Simple UI (sliders/buttons) for users to vote on the 'Safety' of a product: Clean, Sketchy, Wrecked. Use haptic feedback and animations.
- User Voting (Taste): Simple UI (sliders/buttons) for users to vote on the 'Taste' of a product: Yass, Meh, Pass. Use haptic feedback and animations.
- Trending Safe Foods: Display a list of 'Trending Safe Foods' on the landing page.
- Social Sharing (SEO): Generate dynamic Open Graph images/text for shared links with product safety/taste ratings.
- Report Product: Allow users to report products to moderators. Moderators use a protected route to remove the reports. Admin page to remove/resolve them.
- User Account Management: Allow users to create, manage and delete accounts. Store user data securely.
- Admin Account Management: Allow admins to manage user accounts, send emails, and resolve issues.
- Product Image Recognition: Allow users to upload product photos, and use AI to extract the product name from the image.
- Analytics Dashboard: Track product searches, user engagement, and other key metrics for internal use and potential B2B partnerships.
- Ad Slots: Dedicated ad slots within the UI for future monetization.

## Style Guidelines:

- Dark mode background: Very dark grey (#121212) to reduce eye strain.
- Primary color: Vibrant purple (#BF5AF2) to reflect Gen Z/Alpha vibe and contrast against the dark background.
- Accent color: Neon green (#39FF14) for interactive elements, votes, and highlights to emulate a 'VibeCheck' aesthetic.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text. Note: currently only Google Fonts are supported.
- Use minimalist, high-contrast icons that reflect Gen Z/Alpha culture and clearly communicate safety/taste ratings (e.g., fire emojis, ice emojis).
- Mobile-first layout optimized for thumb interactions, with clearly defined 'slots' for future ad placements.
- Subtle animations for user interactions, such as voting and search, to enhance the 'rewarding' tactile feel.
